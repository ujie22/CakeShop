### 檔案說明

# login.php => 會員登入
會員輸入帳密，此帳密會傳入backend.php做檢查，若沒有問題即可登入成功

# backend.php => 會員資料檢查
會員輸入帳密後，會至此做檢查，進行比對帳密
若資料符合:直接登入成功，並且前往商店頁面
若資料不符合:傳至login_inc.php，顯示錯誤，需要重新輸入

# login_inc.php => 會員未輸入帳密or輸入錯的帳密
顯示錯誤訊息，可以點選重新輸入即可回到login.php重新輸入帳密

# shop.php => 商店頁面
含商品的名稱、圖片、價格
登入成功後即可做購買，按右方"放入購物車"就可以將該商品放入購物車(cart.php)，可持續放入
點選下方按鈕"查看購物車"，可以查看購物車內容

# cart.php => 購物車內容
含商品名稱、圖片、價格、已點選數量
下方有兩個按鈕:
繼續購物:返回商店頁面(shop.php)可以繼續做購買
結帳清單:進入結帳頁面(list.php)

# list.php => 結帳清單
含商品名稱、圖片、價格、購買數量、加總金額
下方會顯示購買總金額、會員等級、優惠後的價格
最下方有兩個按鈕:
取消:回到購物清單(cart.php)可以繼續做選購
結帳:進入最後一個頁面(end.php)，完成交易

# end.php => 完成交易
顯示訂單完成並清除購物車(cart.php)內容
下方有重新購買鍵，點選後回到登入頁面(login.php)，需要重新輸入帳密進行登入購買

# product.php => 產品資料
將所有產品寫入，供商店、購物車、結帳清單使用

# customer.txt => 會員資料
含會員帳號、密碼、會員等級
將資料傳入backend.php做會員資料檢查

會員等級   滿1500折扣   沒滿1500
1.VIP        7折          8折
2.Gold       75折         85折
3.Guest      8折          9折
